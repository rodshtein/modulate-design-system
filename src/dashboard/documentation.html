Velma-2 Models

Velma-2 handles transcription, emotion, accent, and engagement detection across 70+ languages. Choose the model that fits your use case.

Velma-2 Batch — best-in-class multilingual batch transcription in 70+ languages with speaker diarization, emotion detection, accent identification, and PII/PHI tagging. Supports AAC, AIFF, FLAC, MOV, MP3, MP4, OGG, Opus, WAV, and WebM up to 100MB. REST API.

Velma-2 Batch English Fast — English-only batch transcription with extremely high throughput and automatic capitalization and punctuation. Opus format audio only. Ideal for high-scale batch processing. REST API.

Velma-2 Streaming — real-time streaming transcription via WebSocket with full voice-native understanding: multilingual support, speaker diarization, emotion, accent, and PII/PHI tagging. WebSocket.

Feature comparison

Transcription — supported in all three models. Speaker diarization — Velma-2 Batch and Streaming. Emotion detection — Velma-2 Batch and Streaming. Accent identification — Velma-2 Batch and Streaming. PII/PHI tagging — Velma-2 Batch and Streaming. Multilingual — Velma-2 Batch and Streaming. Auto capitalization and punctuation — all three models. Real-time — Velma-2 Streaming only.

Authentication & rate limits

REST endpoints require an API key via the X-API-Key header. The WebSocket endpoint uses an api_key query parameter.

Per-model concurrency and monthly usage quotas. Credit-based billing with free tier included. Usage tracked in real time via the Usage Dashboard.
